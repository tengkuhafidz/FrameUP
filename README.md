# frameup

## Project setup
1) create a config.ts file under src, and export it
2) Install dependencies with `yarn install`

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
